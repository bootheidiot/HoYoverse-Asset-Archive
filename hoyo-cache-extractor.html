HoYo Launcher Assets (HYP)

  1. Press Windows + R, type %appdata% and hit Enter.
  2. Navigate to: \Roaming\Cognosphere\HYP\1_0\fedata\Cache\Cache_Data
  3. Drop the data_1 file below. This is the main file for launcher videos and images, but you can upload any other cache files here to browse their contents.
Some players may also have a HYP\1_1\ folder or a HYP\beta\ folder if you have participated in any betas. These folders contain additional data. If you don't find what you're looking for, check the same directory path but replace 1_0 with one of these other folder names.

In-Game Web Assets

  1. Open the HoYo Launcher and click the three lines next to the Start Game button on the game you want to extract from.
  2. Click View Screenshot, then go back one folder to reach the game's root directory.
  3. Navigate to: [Your Game]_Data\webCaches
  4. You can upload an individual cache folder or the entire webCaches folder to browse all events and browser assets at once.
  5. Inside whichever cache folder you choose, go to Cache\Cache_Data and upload the data_1 file. This is the primary location for cached data, but you can upload other files here too.
